Class Selector

Classes can be applied to multiple items on a page. They are ideal for reusing styles on different elements.

Here's how we hook a class attribute to an element:

class="highlight"

Here's how the CSS is applied to the element:

.highlight {
            color: blue;
            font-weight: bold;
        }

Compound Selector

Compound selectors combine more than one selector, specifying multiple conditions. Here, we use a class and an ID together.

Here's an example of a compound selector:

div#id-div.highlight

Here's how the CSS is applied:

div#id-div.highlight {
            border: 2px solid red;
            background-color: yellow;
        }

Descendant Selector

This selector targets all specific descendants within a parent element.

.highlight ul li {
          color: blue;
